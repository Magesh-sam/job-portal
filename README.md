# Job Portal 
**Tech Stack:**  React, Typescript, React Router, React Hook Form, Redux Toolkit, Shadcn UI, Tailwind css, JSON server - to mock API. TinyMCE - WYSIWYG Text Editor
## Features

 - Add new job
 - Edit an existing job
 - Delete a job
 - Apply for a job
 - Filter jobs
 - Search jobs
 - Light/Dark/System Theme Toggle
 - Profile page
